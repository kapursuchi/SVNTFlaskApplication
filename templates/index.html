Welcome to Suchi Kapur's basic Flask app!

The following paths conduct certain tasks:

  1. This path displays #s between 1 and the specified number: /<int:number>
  2. This path displays odd #s between 1 and the specified number: /<int:number>/odd
  3. This path displays even #s between 1 and the specified number: /<int:number>/even
  4. This path displays prime #s between 1 and the specified number: /<int:number>/prime 

        ***Note***
        The imports I used are:
            from flask import *
            from sympy import *

        Both of these required:
            pip install <flask or sympy>
